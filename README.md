# Vue.js + Vuetify App

Aplicación Vue.js 3 con Vuetify 3, diseño responsivo con header, sidebar y menú hamburguesa.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

## Características

- ✅ Vue 3 con Composition API
- ✅ Vuetify 3 UI Framework
- ✅ Vue Router para navegación
- ✅ Layout responsivo con sidebar
- ✅ Menú hamburguesa para móviles
- ✅ Material Design Icons
- ✅ Single-File Components
- ✅ Vite como bundler

## Estructura del Proyecto

```
src/
├── main.js              # Punto de entrada
├── App.vue              # Componente principal con layout
├── style.css            # Estilos globales
├── plugins/
│   └── vuetify.js       # Configuración de Vuetify
├── router/
│   └── index.js         # Configuración de rutas
└── views/
    ├── Home.vue         # Vista de inicio
    ├── About.vue        # Vista acerca de
    └── Settings.vue     # Vista de configuración
```

## Build para Producción

```bash
npm run build
```
